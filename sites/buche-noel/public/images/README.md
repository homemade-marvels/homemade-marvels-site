## Quick Instructions

1. **Copy your cake photos to this folder**
   - Drag and drop them here
   - Or use Copy/Paste from File Explorer

2. **Use descriptive names**
   - Good: `chocolate-birthday-cake.jpg`
   - Bad: `IMG_1234.jpg`

3. **Update the cakes.json file**
   - Location: `sites/generic/data/cakes.json`
   - Change the "image" field to: `/images/your-photo-name.jpg`

4. **Save and refresh your browser** - that's it!
